NZ.UA

General notes

nz.ua is a platform that provides electronic class journals and diaries with features for remote learning.

API detail

authorization

This API endpoint is used for authenticating a user by verifying their login credentials. Upon successful authentication, it returns a token that can be used for subsequent requests requiring user authentication.

  • Curl
  • HTTP
curl -X POST -H "Content-Type: application/json" -H "Accept-Encoding: application/gzip" -H "Connection: Keep-Alive" -H "Accept-Charset: utf-8, *;q=0.8" -d '{
    "password": "password",
    "username": "username"
}' "https://api-mobile.nz.ua/v1/user/login"
POST /v1/user/login HTTP/1.1
Host: api-mobile.nz.ua
Content-Type: application/json
Accept-Encoding: application/gzip
Connection: Keep-Alive

    "password": "password",

student-performance

This API endpoint retrieves the performance data of a student, including the number of missed days and lessons, as well as the list of subjects and their respective performance metrics.
